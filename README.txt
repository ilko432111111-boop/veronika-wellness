# Veronika chatbot fixed files

These files add short-term chat memory so the bot understands replies like:
- "30 minutes?"
- "muscles"
- "yes"
- "how much?"

## What to do

1. Download and unzip this folder.
2. Copy these files into your project folder:
   - chatbot.py
   - index.html
   - requirements.txt
   - start.sh
3. Replace the old files when Windows asks.
4. Open Git Bash in that folder.
5. Run:

git add .
git commit -m "Add chatbot memory and improve replies"
git push

6. Wait for Render to redeploy both services.
7. Test the chatbot again.
